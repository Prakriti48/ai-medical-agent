# 🧠 AI Medical Agent

An AI-powered medical consultation web app that helps users input symptoms, get relevant suggestions, and consult with a virtual doctor. Built with **Next.js**, **Clerk Auth**, **PostgreSQL**, and **Prisma**.

---

## 🚀 Features

- 🩺 AI-based symptom checker
- 👤 Secure login via Clerk authentication
- 📊 Real-time consultation record storage
- 📂 PostgreSQL + Prisma ORM integration
- 🧠 Smart suggestions and AI workflow (via LangChain or LLMs)
- 📋 Prisma Studio for managing data models

---

## 🛠️ Tech Stack

| Tech        | Description                            |
|-------------|----------------------------------------|
| Next.js     | React framework for frontend/backend   |
| Prisma ORM  | Type-safe database ORM (PostgreSQL)    |
| Clerk       | Authentication system                  |
| TailwindCSS | Modern styling                         |
| PostgreSQL  | Relational DB                          |
